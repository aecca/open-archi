Menú contextual

Cada elemento tendrá oportunidad de mostrar un menú contextual dependiendo de sus capacidades específicas. Cada tipo de elemento determinará cuále sacciones se pueden tomar. También influirá la permisología con la que cuenta el usuario autenticado

  • Haga clic con el botón derecho o mantenga presionado un nodo para que aparezca un menú contextual.
  • Si tiene una selección copiada en el portapapeles, puede abrir un menú contextual para pegar.
