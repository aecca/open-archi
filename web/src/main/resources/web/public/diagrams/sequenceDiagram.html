Diagrama de secuencias

Un diagrama de secuencia es un diagrama de interacción que muestra cómo las entidades operan entre sí y en qué orden.

Hay muchos precedentes de soluciones en donde la determinicaicón del orden y el flujo de intearcción entre componentes condicionan el modelo de solución. Esta es una solución clásica utilizando un digrama UML de secuancias convencional para describir, a petición, el flujo de interacción entre los componentes de un modelo.
