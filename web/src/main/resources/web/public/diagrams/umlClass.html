Diagrama de clases

Un clásico. Si fuere necesario, o del interes de algún equipo de desarrollo, sería posible adjuntar el diagrama de clases propuesto para la solución puntual de una pieza específica.

En este ejemplo, los prefijos de símbolos indican la visibilidad de los métodos y las propiedades. Las tres posibilidades son:

  • + (Public)
  • - (Private)
  • # (Protected)
